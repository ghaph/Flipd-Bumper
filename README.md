# [Flipd.gg](https://flipd.gg) Auto Bumper!

### Instructions

1. Put config file next to `bumper.py`
2. Fill out the config.json
3. Install brave browser (optional but recommended)
4. Run `pip install -r requirements.txt`
5. Start bumper.py
